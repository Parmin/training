What is the MMS API
-------------------

Always users to programatically:

- Access monitoring data
- Backup functionality (Request backups, change snapshot schedules, etc.)
- Automation cluster configuration (modify, view)

Sample MMS API Uses Cases
-------------------------

- Ingest MMS Monitoring Data
- Programatically Restore Environments
- Configuration Management

  
Ingest MMS Monitoring Data
--------------------------

The MMS monitoring API can be used to ingest monitoring data into another system, such as Nagios, HP OpenView, or your own internal dashboard
  
Programatically Restore Environments
------------------------------------

Use the MMS backup API to programatically restore an integration testing environment based on the last production snapshot.

Configuration Management
------------------------

Use the MMS automation API to integrate with exisiting configuration management tools (such as Chef or Puppet) to automate creating and mainitaing environments.
