Exercise: Creating a 2d Index
-----------------------------

Create a 2d index on the collection ``testcol`` with:

  - A min value of -20 
  - A max value of 20
  - 10 bits of precision 
  - The field indexed should be ``xy``.

.. only:: instructor

  .. note::

    Answer:

    .. code-block:: javascript

      db.testcol.ensureIndex( { xy : "2d" }, { min : -20, max : 20, bits : 10 } )
