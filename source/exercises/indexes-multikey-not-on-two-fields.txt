Example: Multikey Indexes on Multiple Fields
--------------------------------------------

.. code-block:: javascript

    db.multiKeyLimits.ensureIndex( { x : 1, y : 1 } )
    db.multiKeyLimits.insert( { _id : 1, x : 1, y : 1 } )  // no problems yet
    db.multiKeyLimits.insert( { _id : 2, x : [ 1, 2 ], y : 1 } )  // still OK
    db.multiKeyLimits.insert( { _id : 2, x : 1, y : [ 1, 2 ] } )  // still OK
    db.multiKeyLimits.insert( { _id : 2, x : [ 1, 2 ], y : [ 1, 2 ] } )
    // Won't work
