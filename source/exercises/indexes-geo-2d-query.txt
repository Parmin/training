Exercise: Querying 2d geo Documents
-----------------------------------

Please find all documents within 1, 3, and 6 degrees of the point [ -2.5, -0.5 ].

.. code-block:: javascript

    db.twoD.find( { xy : { $geoWithin : { $center : [ [ -2.5, -0.5 ], 1 ] } } } )  // 1 doc
    db.twoD.find( { xy : { $geoWithin : { $center : [ [ -2.5, -0.5 ], 3 ] } } } )  // 2 docs
    db.twoD.find( { xy : { $geoWithin : { $center : [ [ -2.5, -0.5 ], 6 ] } } } )  // 3 docs, not ordered by distance.
